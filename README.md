# diathiki-vasilissas PROJECT DESCRIPTION

Το παρόν έργο είναι διαδραστικό ψηφιακό παιχνίδι και προορίζεται αποκλειστικά για ψυχαγωγική χρήση.Αποτελεί διαδραστική ψηφιακή εφαρμογή ψυχαγωγικού και εκπαιδευτικού χαρακτήρα (πιλοτική). Δημιουργήθηκε με τη βοήθεια εργαλείων τεχνητής νοημοσύνης. Ορισμένα οπτικοακουστικά στοιχεία που δημιουργήθηκαν με εργαλεία τεχνητής νοημοσύνης τρίτων παρόχων φέρουν τα αντίστοιχα υδατογραφήματα.
Τίτλος έργου:
Η Διαθήκη της Βασίλισσας
Τύπος:
Πιλοτική διαδραστική ψηφιακή εφαρμογή βιωματικής μάθησης (gamification)
Περιγραφή:
Η «Διαθήκη της Βασίλισσας» είναι ένα διαδραστικό ψηφιακό παιχνίδι τύπου treasure hunt, σχεδιασμένο για χρήση σε εξωτερικό χώρο (πάρκο).
Οι συμμετέχοντες παρακολουθούν σύντομα βίντεο-γρίφους, απαντούν ερωτήσεις και σταδιακά αποκαλύπτουν τμήματα ενός ψηφιακού χάρτη, μέχρι την ολοκλήρωση της αποστολής.
Η εφαρμογή λειτουργεί μέσω κινητού τηλεφώνου, χωρίς εγκατάσταση, και αξιοποιεί στοιχεία αφήγησης, εξερεύνησης και συνεργατικής μάθησης.
Το έργο αποτελεί μυθοπλασία, εμπνευσμένη από ιστορικές περιόδους, και δεν αναπαριστά πραγματικά γεγονότα.
Σκοπός:
Η ενίσχυση της βιωματικής εμπειρίας των επισκεπτών του πάρκου μέσω παιχνιδιού, αφήγησης και ενεργητικής συμμετοχής.
Καθεστώς:
Πιλοτική εκπαιδευτική εφαρμογή – μη εμπορική χρήση.
3️⃣ ΜΑΘΗΣΙΑΚΟΙ ΣΤΟΧΟΙ:
Η εφαρμογή στοχεύει:
🎯 Γνωστικοί στόχοι
Εξοικείωση με τον χώρο του πάρκου μέσα από εξερεύνηση
Σύνδεση αφηγηματικών στοιχείων με τον φυσικό χώρο
Ανάπτυξη παρατηρητικότητας και λογικής σκέψης
🧠 Δεξιότητες
Επίλυση προβλημάτων μέσω γρίφων

🌿 Στάσεις & αξίες
Ενίσχυση της ενεργητικής συμμετοχής
Θετική σύνδεση με τον δημόσιο χώρο
Ενθάρρυνση της βιωματικής μάθησης εκτός τάξης
📱 Ψηφιακές δεξιότητες
Χρήση κινητού ως εκπαιδευτικού εργαλείου
Εξοικείωση με ψηφιακά διαδραστικά μέσα
